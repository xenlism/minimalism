Get Installations

You can install Xenlism Minimalism by pacman and Apt-Get Git or WGET and CURl

We have many solution to help you install Xenlism Minimalism to your system by easy.


Debian/Ubuntu Apt-Get Repository


Xenlism Minimalism support Debian/Ubuntu Base apt-get Repositories.

Following Command line in Terminal.

add sourceforg.net public gnupg key to the APT keys database 

	sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 2B80AC38

add ppa repository to apt-get 


	sudo add-apt-repository ppa:xenatt/xenlism

Update System and Install Xenlism Minimalism Gnome/GTK theme. 


	sudo apt-get update
	sudo apt-get install xenlism-minimalism-theme

Arch Linux Pacman Repository


Xenlism Wildfire support Arch Linux Base Pacman Repositories.

Following Command line in Terminal. 


		sudo nano /etc/pacman.conf

add config section to /etc/pacman.conf 


	[xenlism-arch]
	SigLevel = Never
	Server = http://downloads.sourceforge.net/project/xenlism-wildfire/repo/arch

Update System and Install Xenlism Minimalism Gnome/GTK theme. 


	sudo pacman -Syyu
	sudo pacman -S xenlism-minimalism-theme

Other Linux Distributions

Xenlism Wildfire Installations Support A most Linux and *nix System with CURl or WGET and GIT.


Install with wget


	bash <(wget -qO- https://raw.githubusercontent.com/xenlism/minimalism/master/INSTALL/online.install)

Install with curl


	bash <(curl -s https://raw.githubusercontent.com/xenlism/minimalism/master/INSTALL/online.install)

Install with git


	cd /tmp;git clone https://github.com/xenlism/minimalism.git;bash /tmp/minimalism/INSTALL/offine.install;

Install with wget and git


	bash <(wget -qO- https://raw.githubusercontent.com/xenlism/minimalism/master/INSTALL/git.install)

Install with curl and git


	bash <(curl -s https://raw.githubusercontent.com/xenlism/minimalism/master/INSTALL/git.install)
